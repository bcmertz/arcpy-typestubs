[arcpy now includes its own type stubs as of ArcGIS Pro 3.3](https://www.esri.com/arcgis-blog/products/arcgis-pro/developers/typehints-have-arrived-in-arcgis-pro-3-3), so this project is no longer needed, and will no longer be developed.

# arccpy Type Stubs

Initial type stubs generated using the following command:

```cmd
pyright --createstub arcpy
```
